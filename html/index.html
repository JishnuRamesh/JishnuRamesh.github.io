Project

Selenium Framework

Python, Selenium

This is a instance of a python-selenium script under POM model that is perfectly scalable . One the time consuming parts of any selenium job is the need to wait for a specific element to show up in the page. This script takes care of this problem by subclassing wedriver and webelement and takes care of the element loading internally so that you don't have to wait for element loads.
